Make yours look like this

        
    
          
  1. 
            
    Save the config to your home directory:
    
            
    cp tmux.conf ~/.tmux.conf
    
            
    Or use the Download button above, then move the file.
    
          
    2. 
          
  2. 
            
    Install a Nerd Font for the battery icons:
    
            
    # choose one from https://www.nerdfonts.com/font-downloads
# install it, then set it as your terminal font
    
          
    3. 
          
  3. 
            
    Install Starship for the shell prompt:
    
            
    curl -sS https://starship.rs/install.sh | sh
    
            
    Then add the init line to your shell rc file:
    
            
    # zsh, in ~/.zshrc
eval "$(starship init zsh)"

# bash, in ~/.bashrc
eval "$(starship init bash)"
    
          
    4. 
          
  4. 
            
    Reload tmux (or start a fresh server):
    
            
    tmux source-file ~/.tmux.conf
    
          
    5.
